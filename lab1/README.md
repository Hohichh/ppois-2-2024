# Отчет о лабораторной работе № 1 
# "Модель стрелкового оружия"

## Структура проекта

Проект состоит из следующих модулей:

1. **gushop**: Модуль, содержащий класс 'Gunshop', представляющий собой оружейный магазин,
предоставляющий возможность покупки и продажи оружия игроком.
2. **player**: Модуль, содержащий класс 'Player', представляет собой игрока
и позволяет пользователю взаимодействовать с классом оружейного магазина. 
3. **weapon**: Модуль, включающий в себя класс 'Weapon', представляющий непосредственно модель стрелкового оружия.
4. **cartridge**: Модуль, который содержит класс 'Cartridge', являющийся моделью патрона.
5. **magazin**: Модуль, содержащий класс 'Magazin', который представляет собой магазин(обойму).
6. **barrel**: Модуль, реализующий класс 'Barrel', который представляет оружейный ствол.
7. **trigger**: Модуль, который реализует класс 'Trigger', то есть спусковой механизм.
8. **aim**: Модуль, содержащий класс 'Aim', для прицела.

## Описание классов

### gushop
Класс содержащий 2 словаря: `weapons` и `cartridges`. Словари содержат строковые ключи с названиями типов
оружия, по каждому ключу находится список. В `weapons` находятся списки объектов класса `Weapon`, а в `cartridges` содержатся
списки со списками объектов типа `Cartridge`. Реализует механики покупки и продажи оружия.
    
**Поля:**
- `self.__weapons`
- `self.__cartridges`
- `self.__budget`

**Методы:**
- `buy_weapon(self, weapon: Weapon)`
- `buy_cartridges(self, cartridges: List[Cartridge])`
- `sell_weapon(self, weapon: Weapon) -> Weapon`
- `sell_cartridges(self, cartridges: List[Cartridge]) -> List[Cartridge]`
- `show_assortment(self)`
- `add_weapon(self, weapon: Weapon, weapon_type: str)`
- `add_cartridges(self, cartridges: List[str], weapon_type: str)`
- `find_weapon(self, name: str) -> Weapon`
- `find_cartridges(self, cartridge_name: str) -> List[Cartridge]`

### Класс Weapon

Класс `Weapon` представляет оружие. Он содержит следующие атрибуты:

- `name` (str): Имя оружия.
- `weapon_type` (str): Тип оружия.
- `magazine` (Magazine, по умолчанию None): Объект типа `Magazine`, представляющий магазин оружия.
- `aim` (Aim, по умолчанию None): Объект типа `Aim`, представляющий прицел оружия.
- `barrel` (Barrel, по умолчанию None): Объект типа `Barrel`, представляющий ствол оружия.
- `price` (float, по умолчанию 2): Цена оружия.
- `grime` (float): Уровень загрязнения оружия.

#### Методы:

- `__init__(self, name: str, weapon_type: str, magazine: Magazine=None, aim: Aim=None, barrel: Barrel=None, price: float=2)`: Инициализирует объект класса Weapon с заданными параметрами.
- `__str__(self)`: Возвращает строковое представление оружия.
- `reload(self, cartridges: List[Cartridge])`: Перезаряжает оружие.
- `unload(self)`: Разгружает магазин оружия.
- `take_aim(self, rotation_angle: float)`: Наводит прицел на цель с заданным углом поворота.
- `shoot(self)`: Выстреливает из оружия.
- `maintenance(self, operation: int)`: Проводит техническое обслуживание оружия.
- `safe_storage(self) -> bool`: Проверяет, находится ли магазин оружия в пустом состоянии.

#### Свойства:

- `name`: Геттер и сеттер для имени оружия.
- `weapon_type`: Геттер и сеттер для типа оружия.
- `magazine`: Геттер и сеттер для магазина оружия.
- `aim`: Геттер и сеттер для прицела оружия.
- `barrel`: Геттер и сеттер для ствола оружия.
- `price`: Геттер и сеттер для цены оружия.
- `grime`: Геттер для уровня загрязнения оружия.

#### Свойства:

# Класс "Magazine"

## Описание

Класс "Magazine" представляет собой модель магазина для огнестрельного оружия. Магазин хранит патроны определенного типа и имеет свойства, такие как емкость, тип оружия, калибр и состояние (пустой или заполненный). Кроме того, магазин может загружать и выгружать патроны, а также предоставлять информацию о содержимом.

## Атрибуты

- `name`: Название магазина (строка)
- `weapon_type`: Тип оружия, для которого предназначен магазин (строка)
- `capacity`: Емкость магазина (целое число)
- `caliber`: Калибр патронов в магазине (вещественное число)
- `cartridges`: Список патронов в магазине (список объектов типа "Cartridge")
- `empty`: Флаг, указывающий на пустоту магазина (логическое значение)

## Методы

- `__init__(self, name: str, weapon_type: str, capacity: int, caliber: float, cartridges: List[Cartridge]=None)`: Конструктор класса, инициализирует атрибуты магазина.
- `__str__(self)`: Возвращает строковое представление магазина.
- `load_cartridge(self, cartridge: Cartridge)`: Загружает патрон в магазин.
- `unload(self)`: Выгружает все патроны из магазина.
- `cartridge_feed(self) -> Cartridge`: Извлекает патрон из магазина.
- `name(self)`: Получает название магазина.
- `empty(self)`: Проверяет, пуст ли магазин.
- `capacity(self)`: Получает емкость магазина.
- `weapon_type(self)`: Получает тип оружия магазина.
- `caliber(self)`: Получает калибр патронов в магазине.
- `cartridges(self)`: Получает список патронов в магазине.


# Класс "Barrel"

## Описание

Класс "Barrel" представляет собой модель ствола огнестрельного оружия. Ствол имеет атрибуты, такие как название, тип оружия, калибр, состояние (разрезан или нет) и список патронов. Ствол может быть загружен патроном и производить выстрел.

## Атрибуты

- `name`: Название ствола (строка)
- `weapon_type`: Тип оружия, для которого предназначен ствол (строка)
- `caliber`: Калибр ствола (вещественное число)
- `is_cut`: Флаг, указывающий на то, разрезан ствол или нет (логическое значение)
- `cartridges`: Список патронов в стволе (список объектов типа "Cartridge")

## Методы

- `__init__(self, name: str, weapon_type: str, caliber: float)`: Конструктор класса, инициализирует атрибуты ствола.
- `__str__(self)`: Возвращает строковое представление ствола.
- `load_barrel(self, cartridge: Cartridge)`: Загружает патрон в ствол.
- `fire(self)`: Производит выстрел из ствола.
- `name(self)`: Получает название ствола.
- `weapon_type(self)`: Получает тип оружия ствола.
- `caliber(self)`: Получает калибр ствола.
- `is_cut(self)`: Получает информацию о том, разрезан ли ствол.
- `cartridges(self)`: Получает список патронов в стволе.


# Класс "Trigger"

## Описание

Класс "Trigger" представляет собой модель элемента огнестрельного оружия, который отвечает за инициирование выстрела. Этот класс содержит метод, который позволяет нажать на спусковой крючок, производя выстрел из оружия.

## Методы

- `pull(self, magazine: Magazine, barrel: Barrel)`: Метод, который инициирует выстрел. Принимает в качестве параметров объекты магазина и ствола. Если магазин пуст, выводит сообщение о том, что оружие не может стрелять из-за отсутствия патронов. В противном случае выводит звук выстрела и загружает патрон из магазина в ствол, а затем производит выстрел из ствола.


# Класс "Aim"

## Описание

Класс "Aim" представляет собой модель элемента огнестрельного оружия, который используется для наведения и прицеливания. Прицел может быть различного типа, такого как оптический или другие типы, и имеет свойства, такие как название, тип оружия, тип прицела, увеличение (для оптического прицела) и угол поворота.

## Атрибуты

- `name`: Название прицела (строка)
- `weapon_type`: Тип оружия, для которого предназначен прицел (строка)
- `aim_type`: Тип прицела (строка)
- `magnification`: Увеличение (только для оптического прицела) (вещественное число)
- `angle`: Угол поворота прицела (в градусах)

## Методы

- `__init__(self, name: str, weapon_type: str, aim_type: str, magnification: float=0)`: Конструктор класса, инициализирует атрибуты прицела.
- `__str__(self)`: Возвращает строковое представление прицела.
- `rotate(self, delta: float)`: Поворачивает прицел на заданный угол.
- `name(self)`: Получает название прицела.
- `weapon_type(self)`: Получает тип оружия, для которого предназначен прицел.
- `aim_type(self)`: Получает тип прицела.
- `magnification(self)`: Получает увеличение прицела.
- `angle(self)`: Получает угол поворота прицела.
